Totality PS

Securely print & scan from the web

Totality PS allows secure printing and scanning from the web. Websites and web applications can now have the reliability and quality of PDF printing & scanning without special plug-in or driver software, thereby supporting printing & scanning from PCs, tablets and mobile devices.

Download

User Guide

Print Features

  • Print high quality PDF files directly to a printer
  • Print Microsoft Word and image files in high quality PDF format (Totality Content Server required)
  • Print from mobile and tablet devices such as the iPhone, iPad & Android phones
  • Initiate and control printing from web page
  • Adobe Acrobat and other 3rd party software not required on client or server
  • Print status messages are sent back to the web site and can be displayed to the user as desired
  • Web pages can specify # of copies & other print settings
  • Paper tray selection can be made on a page-by-page basis to print certain pages on letterhead, colored paper and so forth
  • Website users don't need to install any printer drivers
  • Print to remote printers
  • Websites can control which printers are visible to the user
  • Websites can control which documents can be printed on a user-by-user basis
  • Track and record information about each print job

Scan Features

  • Scan paper to small, high quality PDF or Secure DOCX files using our patented TEngine technology. Reduce file sizes by 99% while maintaining perfect visual quality
  • Works with any TWAIN-enabled scanner or copier
  • Add existing PDF or Microsoft Word content to scanned documents
  • Scan from mobile and tablet devices such as the iPhone, iPad & Android phones
  • Website users don't need to install any scanner drivers
  • Web app users can scan from remote scanners
  • Adobe Acrobat and other 3rd party software not required on client or server
  • Websites can control which scanners are visible to the user
  • Track and record information about each scan job

Specifications

Totality PS runs on a Windows PC that is on the same local area network as the printers and scanners to be used by the web site.

System Requirements

  • Operating System: Windows XP, Windows Vista, Windows 7, Windows Server 2003/2008
  • Memory: 1 GB min, 2 GB recommended
  • Scan: TWAIN scan drivers
  • Print: Printer drivers
